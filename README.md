# AutoMLWebApp

### Using Streamlit ML Web App ###

1. Step 1:
   - Clone the repository
     ```
     git clone "https://github.com/SriArunM/AutoMLWebApp.git"
     ```

2. Step 2:
   - Install requirements
     ```
     pip install -r requirements.txt
     ```

3. Step 3:
   - Run app.py
     ```
     streamlit run "path/to/app.py"
     ```
